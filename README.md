# Todo-List (PHP + HTML/CSS/JS)

Projeto simples de gerenciador de todo-lists com autenticação. 
 
Objetivo: aprender integração front-end / back-end com CRUD de cards (categorias) e tarefas.


## Visão geral
- Login por usuário (sessão PHP).
- Home com cards (cada card = uma lista / categoria).
- Cada card possui: título, descrição e várias tarefas (checkbox).
- Card “+” para criar novo card.
- CRUD completo: criar/editar/deletar cards e tarefas; marcar tarefas como concluídas.
- Banco de dados: MySQL (padrão) — pode adaptar para SQLite ou JSON para protótipo.
